Overview

The user manual maker is a module that allows you to create process manuals for users, it is intended to facilitate the training of new collaborators either in specific tasks or new processes.

Features

Create multiple user manuals
Create manual classifications
Create sections
Reorder steps and sections
Have quality control with an approval request
Easily train new staff
